Kurulum (Ücretsiz Yayınla)

Seçenek A — GitHub Pages

  1. GitHub hesabı aç → github.com
  2. Yeni repo oluştur: yigit-site
  3. Bu dosyaları repoya yükle
  4. Settings → Pages → Source: Deploy from a branch, Branch: main, / (root)
  5. 3–5 dk içinde siten şu adreste olur: https://kullaniciadın.github.io/yigit-site/

Seçenek B — Netlify

  1. Netlify hesabı aç → netlify.com
  2. New site from Git → GitHub repo’nu bağla
  3. Build komutu gerekmiyor; direkt yayınlar
  4. Ücretsiz *.netlify.app alt alan adı alırsın

Özel alan adı istersen alan adının ücreti olur; barındırma yine ücretsiz kalır.
